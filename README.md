# Spending
### Needs vs Wants

#### Project was built with the tools mentioned below and the purpose of the project is to give some perspective on someone's spending if they have more impulse buying than anything else. 
#### Seeing spending listed out gives a different perspective and may also make someone realize that they need to think before making certain purchases that are not considered a need.
